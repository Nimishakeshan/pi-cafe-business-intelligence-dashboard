# pi-cafe-business-intelligence-dashboard
This Power BI project analyzes Pi-Cafe’s business performance across sales, customer behavior, product performance, and financial metrics. The dashboards provide interactive insights to help management make data-driven decisions.

1. Sales Dashboard:
The Sales Analysis Report provides a comprehensive view of the sales performance for
Pi-Cafe. This report is interactive, allowing users to filter data by year, quarter, month, region,
store, category, item, and order type. The report includes key performance indicators (KPIs),
visualizations of sales trends over time,sales by region, sales by category, Top 5 Item sales,
sales by order type, and breakdowns by category, region, item, and order type.

2. Customer Dashboard:
The Customer Analysis Report provides an in-depth look at customer-related metrics and
ratings for Pi-Cafe. This interactive report allows users to filter data by year, quarter, month,
region, store, category, item, and order type. The report includes key performance indicators
(KPIs), visualizations of customer trends, and distribution of ratings.

3. Financial Analysis Dashboard:
The Financial Analysis Dashboard provides an in-depth overview of PI Café's financial
performance. This dashboard is designed to help CXO, Regional and Store Managers to
understand key financial metrics such as total sales, total expenses, cost of goods sold
(COGS), and net profit margin.

4. Order Dashboard:
The Order Analysis Dashboard provides a comprehensive overview of sales and order metrics
for PI Café. This dashboard is designed to help CXO, Regional and Store Managers to
understand key performance indicators (KPIs) related to Total Sales, Total Orders, Average
Order Value, and Order Frequency per Unique Customer.

5. Product Dashboard:
The Product Analysis Dashboard provides insights into PI Cafe product performance, including
gross profit, customer satisfaction, and sales metrics. This dashboard is designed to help CXO,
Regional and Store Managers to identify high-performing products and areas for improvement.
